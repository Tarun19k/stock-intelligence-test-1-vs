## When to apply themes

### Landing page (`index.html`)
Theme: GSI Midnight Galaxy custom
Apply to: hero section, feature cards, testimonials, footer

### Pitch deck (`GSI_pitch.pptx`)
Theme: Midnight Galaxy → slide master
Accent slides: Tech Innovation highlights for signal/data moments

### PDF reports
Theme: GSI Midnight Galaxy → reportlab StyleSheet
Header: navy background, white title text
Body: light background (`#f5f7fa`) with navy text for print readability

### Excel exports
Theme: Professional dark header + light data rows
Header row: `#0d1b2a` fill, `#4fc3f7` text
Data rows: alternating `#ffffff` / `#f0f4f8`

## Custom theme generation
If existing 10 themes don't fit a specific artifact:
1. Describe the artifact purpose and audience
2. Generate: primary, secondary, accent, text colours + font pairing
3. Confirm + apply consistently across all artifact elements
